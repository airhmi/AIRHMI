# ListViewDeleteX Özelliği


ListView'de istedğimiz satırı bu komut ile silebiliriz. 

```
ListViewDeleteX("ListView1" ,1 );  // 1. satırı sil.
```

ListView nesnesi statik dir. Her sayfadan erişilibilinir.
